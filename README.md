# Customer Behavior Analytics

A complete end-to-end Data Analytics project that analyzes customer shopping behavior using Python, SQL, PostgreSQL, and Power BI. The project focuses on transforming raw data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL queries, and an interactive dashboard.

---

## Project Overview

This project follows the complete data analytics workflow:

- Data Collection
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL Analysis using PostgreSQL
- Interactive Dashboard Development in Power BI
- Business Insights & Recommendations

The goal is to understand customer purchasing behavior, revenue trends, customer demographics, and product performance.

---

## Dataset

The dataset contains customer shopping information, including:

- Customer Details
- Age Group
- Gender
- Product Category
- Purchase Amount
- Review Rating
- Subscription Status
- Shipping Type
- Payment Method
- Purchase Frequency

## Project Workflow

### 1. Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected data types
- Cleaned inconsistent values

### 2. Exploratory Data Analysis (EDA)
- Customer demographics
- Revenue analysis
- Category-wise sales
- Review rating analysis
- Purchase behavior analysis
- Subscription insights

### 3. SQL Analysis (PostgreSQL)
Performed SQL queries to analyze:
- Total Revenue
- Average Purchase Amount
- Customer Segmentation
- Category Performance
- Subscription Analysis
- Age Group Analysis
- Top Products

### 4. Power BI Dashboard
Created an interactive dashboard with filters and KPIs to visualize customer behavior and business performance.

---
## Key Insights

- Clothing generated the highest revenue and sales.
- Young Adults contributed the highest revenue.
- Most customers were non-subscribers.
- Average purchase amount was approximately **$59.8**.
- The average customer review rating was **3.75**.
