<h1>Customer Behaviour Dashboard</h1>
<h3>The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence by:</h3>
<ol><li>Data Preparation,Modeling & Exploratory Data Analysis (Python): Clean and transform the raw dataset for analysis.</li>
  <li>Data Analysis (SQL): Simulate business transactions, and run queries to extract insights on customer segments, loyalty, and purchase drivers.</li>
  <li>Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.</li>
</ol>
